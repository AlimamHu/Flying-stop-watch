# 🚀 Flying Stopwatch Pro: Ultra-High Performance Edition

A high-end, floating productivity stopwatch built for elite performance. Originally designed in Python and now fully optimized with a **Native C++ GDI+ Core Engine**.

## ⚡ The C++ Edge: Extreme Speed
The transition to C++ has transformed this from a simple widget into a high-performance system tool:
- **Zero CPU Impact**: Optimized rendering loop ensures near-zero CPU usage even with 60FPS animations.
- **Ultra-Low Latency**: Native Win32 event handling provides instantaneous response to clicks and hovers.
- **Hardware Accelerated**: Uses GDI+ Hardware acceleration for smooth anti-aliased transparency and glass effects.
- **Minimal RAM**: Consumes less than 10MB of memory, making it perfect for long-running work sessions.

## 🌟 Premium Features

### 🌊 Liquid Mode (Compact)
Toggle into **Liquid Mode** to transform the stopwatch into a minimalist glowing capsule. 
- **Animated Sloshing**: Real-time sine-wave liquid physics.
- **Urgency Sensing**: Liquid color shifts to orange/red as your timer approaches zero.
- **Overtime Filling**: When you exceed your limit, the capsule starts filling back up with "Overtime Juice" in a 2-minute breathing cycle.

### 🌓 Dynamic Design System
- **8 Pro Themes**: Midnight, Nord, Cyberpunk, Forest, Dracula, and more.
- **Rotating Gradient Borders**: A smooth "Lava Lamp" border that pulses with your focus intensity.
- **Glassmorphism**: Transparent, frosted backgrounds that look premium on any wallpaper.

### 🤖 Smart Productivity Mode
- **Auto-Snap**: Automatically detects if you are playing a video or in fullscreen, and snaps the stopwatch to the nearest corner to stay out of your way.
- **Wiggle Reminders**: Every 5 minutes, the stopwatch subtly wiggles to remind you of the time and prevent eye strain.
- **Work Intensity Tracking**: The UI changes color and the pet (`😺` → `🔥`) evolves based on your session length.

### 🌈 Full-Screen Motivation
Every minute, a semi-transparent blur overlay takes over with **Rainbow Animated Quotes** to keep you pushed and focused on your goals.

- **Language**: C++ (Native Win32 API)
- **Graphics**: GDI+ (Transparent Layered Windows)
